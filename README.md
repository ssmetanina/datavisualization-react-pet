# Interactive Scrollytelling Landing (React)

Интерактивный лендинг с прокруткой и сторителлингом, построенный вокруг дата-визуализаций.  
Проект исследует стоимость «взрослой жизни» в России: работу, аренду жилья и базовые расходы.

## О проекте

- **Графики и тексты** были подготовлены в рамках учебного проекта по дисциплине *«Датавизуализация»* (НИУ ВШЭ)
- **Веб-сайт, архитектура, анимации и UX** — полностью моя инициатива и реализация
- Цель проекта — превратить статичные графики в интерактивный пользовательский опыт

Проект выполнен в формате **сторителлинга**: пользователь постепенно погружается в историю по мере прокрутки страницы.

---

## Стек

- **React**
- **JavaScript**
- **CSS (без UI-фреймворков)**
- **Intersection Observer API**
- **Vercel** (деплой)
- Встраивание графиков через **Datawrapper**

> При работе с CSS и анимациями я осознанно использовала AI-инструменты как вспомогательный инструмент — для ускорения итераций, подбора решений и рефакторинга.

---

## Реализованная логика

- Модальное стартовое окно с управлением фокусом и клавиатурой
- Плавное появление секций при скролле (Intersection Observer)
- Навигация по секциям через **pagination dots**
- Отслеживание активной секции при прокрутке
- Адаптивная верстка под desktop и mobile
- Доступность:
  - `aria-labels`
  - `role="dialog"`
  - поддержка `prefers-reduced-motion`
  - фокус-стили для интерактивных элементов
